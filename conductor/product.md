# 產品定義: MD2PPT-Evolution

## 1. 願景與核心價值 (Vision)
**MD2PPT-Evolution** 致力於打破「內容創作」與「視覺呈現」之間的壁壘。我們將 Markdown 的寫作極致效率與 PowerPoint 的專業發表能力完美融合，實現真正的 **「簡報即程式碼 (Presentation as Code)」**。

我們解決的核心痛點：
- **消除排版焦慮**：透過智慧佈局引擎與自動化美化，讓使用者不再浪費時間調整字體與間距。
- **數據視覺化自動化**：將純文字表格轉化為原生可編輯圖表，告別繁瑣的截圖流程。
- **隱私與安全**：全客戶端處理架構，確保商業機密與個人筆記永不離開使用者的電腦。

## 2. 目標用戶 (Target Audience)
- **技術講者與開發者**：需要將技術文件、教學筆記快速轉化為具備程式碼高亮與架構圖的專業簡報。
- **教育工作者與學術研究員**：追求極速的內容產出效率，並需要一致且精美的教學投影片。
- **商務經理人**：希望在熟悉的文字編輯器中整理邏輯，一鍵生成具備原生數據圖表的商務報告。

## 3. 核心功能矩陣 (Core Features)

### 3.1 革命性的寫作工作流
- **混合配置系統**：創新的 `===` (分頁) 與 `---` (配置) 分層語法，完美分離內容與樣式設定。
- **快捷動作側欄 (v0.5+)**：視覺化插入模板，大幅降低 Markdown 語法的記憶成本。
- **斜線指令系統 (v0.8.1+)**：輸入 `/` 即可喚起懸浮選單，快速插入包含範例數據的佈局、圖表與元件，實現流暢的寫作體驗。
- **直覺式資產管理**：支援圖片拖放自動轉 Base64 嵌入，並可直接拖曳設定分頁背景圖。
- **大綱模式與拖拽重排 (v0.14.1)**：新增側欄大綱分頁與預覽區把手，支援直覺的投影片順序調整，系統自動同步更新 Markdown 原始碼。
- **PWA 離線工作流 (v0.14.2)**：支援應用程式安裝與全離線編輯、預覽，確保在無網路環境下依然具備完整的簡報生產力。

### 3.2 專業級排版引擎
- **多元佈局系統**：內建 Grid (網格)、Center (居中)、Quote (引用)、Alert (告警) 等多種經設計師調校的版面。
- **智慧分欄邏輯**：Grid 佈局採用「順序填充」算法，確保標題與關聯內容始終保持一致。
- **動態視覺設定**：每張投影片均可獨立定義背景色、背景圖與過場動畫 (Fade, Slide, Zoom)。

### 3.3 技術內容深度整合
- **原生可編輯圖表 (v0.6+)**：支援將 Markdown 表格直接生成 PPTX 內建圖表 (Bar, Line, Pie, Area)，匯出後仍可在 PowerPoint 編輯數據。
- **技術優先顯示**：具備高品質語法高亮 (Syntax Highlighting) 的程式碼區塊，且在 PPT 中保持可編輯文字格式。
- **專業表格樣式**：所有表格預設套用 Modern Style，具備主題色表頭與斑馬紋底色。
- **角色對話模式**：專為技術模擬與劇本設計的對話渲染器。
- **演講者備忘錄系統 (v0.7+)**：支援在預覽中切換顯示，並同步匯出至 PPTX 的原生備忘錄欄位。
- **Web 演講模式與遙控 (v0.12+)**：將靜態網頁轉化為強大的展演工具。
    - **雙螢幕演講者視圖 (Dual-Screen Console)**：利用 `BroadcastChannel` API。主螢幕全螢幕播放，演講者螢幕顯示「當前頁」、「下一頁預覽」、「備忘錄」與「計時器」。
    - **P2P 手機遙控器 (WebRTC Remote)**：掃描 QR Code，手機變成遙控器（上一頁/下一頁/黑屏）。使用 `PeerJS` 進行瀏覽器對瀏覽器連線，無需後端伺服器。

### 3.4 極致預覽體驗 (WYSIWYG)
- **實時雙向互動 (v0.9+)**：編輯器與預覽區高度同步。使用者可直接點擊預覽區元素觸發 Visual Tweaker，針對文字、圖片與圖表數據進行直覺式調整，系統會自動將修改結果精確回寫至 Markdown 原始碼。
- **動態背景渲染 (v0.10+)**：支援 Mesh Gradient 等生成式背景，預覽區透過 SVG 即時渲染，並確保匯出至 PPTX 時維持高度視覺一致性。
- **智慧同步捲動**：確保編輯位置與預覽畫面始終保持一致。
## 5. 未來展望 (Future Roadmap)
- **主題預設值管理器**：一鍵切換全域配色與風格模板。
- **多格式輸出鏈**：擴展至 PDF、HTML 互動簡報 (Reveal.js) 等多重載體。
