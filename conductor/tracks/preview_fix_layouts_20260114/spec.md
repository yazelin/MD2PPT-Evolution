# 軌道規格書: 預覽修復與專業排版豐富化

## 概述
本軌道的目標是修復目前網頁預覽與 PPT 匯出在「分頁邏輯」上的不一致問題，並引入「投影片佈局 (Layouts)」與「自定義背景色」等功能，提升匯出投影片的專業感與豐富度。

## 功能需求
1.  **邏輯同步與分頁修復**:
    - 統一網頁預覽與 PPT 生成器的 `splitBlocksToSlides` 邏輯。
    - 在網頁預覽介面 (Preview Pane) 中，將內容區分為明顯的「投影片卡片」，模擬實際分頁效果。
2.  **自定義背景色**:
    - 支援透過 YAML Frontmatter 設定全域背景色。
    - 支援在 `---` 分頁線旁使用參數（如 `--- { bg: "#D24726" }`）設定該頁特定顏色。
3.  **專業排版佈局 (Advanced Layouts)**:
    - 支援透過屬性標記（如 `<!-- layout: title-only -->`）切換以下佈局：
        - **雙欄佈局 (Two Columns)**：左右並排的內容。
        - **大數字/強調頁 (Impact)**：置中超大文字排版。
        - **全螢幕背景 (Full Background)**：背景圖片填滿。
        - **加強版對話模式**：精緻的對話方塊視覺效果。
4.  **設計風格**:
    - 採取 **極簡專業感**，維持商務水準的優雅。

## 驗證標準 (Acceptance Criteria)
- [ ] 網頁預覽畫面的分頁與匯出的 PPT 投影片數量、內容完全對應。
- [ ] 使用者能成功為特定投影片指定背景顏色。
- [ ] 指定為 `two-column` 的 Markdown 內容能在 PPT 中正確呈現為左右兩欄。
- [ ] 匯出的 PPT 維持極簡且專業的視覺質感。
