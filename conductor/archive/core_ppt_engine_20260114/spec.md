# 軌道規格書: 核心 PPT 生成引擎實作 (MVP 功能)

## 概述
本軌道的目標是將 `services/pptGenerator.ts` 從骨架轉化為一個功能完備的轉換引擎。它將接收解析後的 `ParsedBlock[]`，並根據 MD2PPT-Evolution 的產品準則，產生符合規格的 PowerPoint 簡報。

## 核心需求
1.  **投影片智慧分割**: 
    - 支援 H1/H2 標題自動分頁。
    - 支援 `---` (水平分隔線) 強制分頁。
2.  **內容渲染**:
    - 標題與內文列表的基本排版（字體、大小、邊距）。
    - **程式碼高亮**: 透過 `pptxgenjs` 渲染帶有背景色與語法顏色（或區塊模擬）的程式碼區塊。
    - **Base64 圖片支援**: 確保 Markdown 中的圖片能正確嵌入。
3.  **視覺風格**:
    - 整合橘紅品牌色 (Orange-Red) 作為主題色。
    - 支援 16:9 等投影片比例。
4.  **特殊功能**:
    - 實作「角色對話模式」的基礎轉換邏輯。

## 技術約束
- 使用 `pptxgenjs` 作為匯出引擎。
- 採用異步處理流程，避免大規模轉換時卡死瀏覽器。
- 遵循 TDD 流程，確保轉換邏輯的穩定性。
