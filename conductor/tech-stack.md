# 技術堆疊: MD2PPT-Evolution

## 核心技術
- **程式語言**: TypeScript
- **前端框架**: React 19
- **構建工具**: Vite 6
- **CSS 框架**: Tailwind CSS

## 關鍵功能庫
- **PowerPoint 生成**: `pptxgenjs` (核心引擎)
- **Markdown 解析**: `marked` (AST 解析)
- **配置驅動**: `js-yaml` (YAML Frontmatter 解析)
- **圖表渲染**: `mermaid` (SVG 預覽與轉換), `recharts` (React 圖表預覽)
- **多國語言**: `i18next`
- **UI 圖標**: `lucide-react`

## 測試與品質保證
- **單元測試**: Vitest
- **元件測試**: React Testing Library
