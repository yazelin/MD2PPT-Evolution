## 1. 快速入門：您的第一份簡報

MD2PPT 的核心概念很簡單：**左邊寫內容，右邊看結果**。

### 基礎結構
一份標準的簡報由三個部分組成：
1.  **全域設定**：放在文件最開頭，定義整份簡報的主題。
2.  **分頁符號**：使用 `===` 來區隔每一張投影片。
3.  **頁面設定**：每頁開頭的 `---` 區塊，定義該頁的版型。

**👇 試著複製貼上這段代碼：**

```markdown
---
title: "我的第一份簡報"
theme: academic
---

# 大家好！
這是我用 MD2PPT 做的第一張投影片。

===

---
layout: two-column
---

## 為什麼選擇 MD2PPT?
- **速度快**: 不用調格式
- **美觀**: 內建專業主題
- **靈活**: 支援 Markdown

:: right ::

## 右邊放張圖
![Logo](https://raw.githubusercontent.com/eric861129/MD2PPT-Evolution/main/public/logo.svg)
```

## 2. 介面導覽：認識工作區

- **📝 左側編輯區**: 您的創作畫布。支援 Markdown 語法高亮與行號顯示。
- **👁️ 右側預覽區**: 即時呈現最終的 PPTX 效果。您可以直接點擊上面的文字或圖片進行修改。
- **⚡ 左側快捷欄 (Sidebar)**:
    - `≡` **目錄**: 快速插入各類模板。
    - `🎨` **主題**: 切換配色風格。
    - `⚙️` **設定**: 調整字體大小或顯示選項。
- **🚀 頂部工具列 (Header)**:
    - **Present**: 進入全螢幕演講模式。
    - **Export PPTX**: 下載您的簡報檔案。
    - **Images**: 將所有投影片匯出為圖片包。

## 3. 視覺魔法：主題與背景

### 全域主題管理器 (Theme Manager)
無需修改任何代碼，一鍵切換整份簡報的風格。點擊左側欄的 **調色盤圖示 (Palette)**：

*   **🏢 商務與專業**: 推薦 `Classic Amber` (經典) 或 `Tech Blue` (科技)。
*   **🔥 活力與熱情**: 推薦 `Sunset Glow` (暖色調) 或 `Coral Vivid` (鮮明)。
*   **🌿 自然與健康**: 推薦 `Fresh Mint` (清新) 或 `Deep Forest` (深邃)。
*   **⬜ 極簡與現代**: 推薦 `Clean Slate` (極簡) 或 `Dark Matter` (深色模式)。

### 生成式背景 (Generative Mesh)
想讓簡報更獨特？試試自動生成的流體漸層背景。

**在頁面設定中加入：**
```yaml
bg: mesh
mesh:
  colors: ["#0F172A", "#3B82F6", "#DB2777"]
  seed: 12345
```
*   **`colors`**: 選擇您喜歡的顏色組合。
*   **`seed`**: 改變數字，隨機生成不同的流動紋理。

💡 **小技巧**：直接點擊預覽區的背景，使用 **Visual Tweaker** 的「Re-roll」按鈕，可以快速嘗試不同的隨機效果！

## 4. 排版大師：佈局與結構

MD2PPT 提供了多種響應式佈局，讓您專注內容，不用煩惱排版。

| 佈局名稱 | 參數值 (`layout`) | 適用場景 | 特色 |
| :--- | :--- | :--- | :--- |
| **標準** | `default` | 內文頁 | 標題在左上，內容自動流動。 |
| **置中** | `center` | 封面、金句 | 內容垂直水平置中，字體自動放大。 |
| **衝擊** | `impact` | 重點強調 | 極大字體，適合只有一句話的頁面。 |
| **雙欄** | `two-column` | 對比、圖文並茂 | 自動分為左右兩欄。使用 `:: right ::` 強制換欄。 |
| **網格** | `grid` | 多圖展示 | 自動排列成 2~4 欄。需搭配 `columns: 3` 參數。 |
| **引用** | `quote` | 名人名言 | 自動添加裝飾性大引號，字體套用襯線體。 |
| **告警** | `alert` | 警告、注意事項 | 帶有醒目的邊框與背景色塊。 |

**範例：使用網格佈局**
```yaml
---
layout: grid
columns: 3
---
## 產品特色
- 特色 A
- 特色 B
- 特色 C
```

## 5. 數據視覺化：圖表與表格

將枯燥的表格瞬間轉化為專業的可視化圖表。

### 支援圖表類型
使用 `::: chart-type` 語法包裹表格即可。

- `chart-bar`: 長條圖 (比較數據)
- `chart-line`: 折線圖 (趨勢變化)
- `chart-pie`: 圓餅圖 (佔比分析)
- `chart-area`: 面積圖 (累積量)

**語法範例：**
```markdown
::: chart-pie { "title": "2026 營收分析", "showLegend": true }

| 產品線 | 佔比 |
| :--- | :--- |
| 軟體   | 60   |
| 硬體   | 30   |
| 服務   | 10   |

:::
```
*注意：JSON 屬性必須使用雙引號 `"`。*

## 6. 效率神器：指令與微調工具

### 🔮 Visual Tweaker (視覺化調整)
覺得改 Markdown 太慢？直接動手吧！
1.  **點擊** 預覽區的任何文字、圖片或圖表。
2.  **彈出面板** 會自動出現。
3.  **直接修改** 內容、字體大小或圖片網址。
4.  系統會 **自動回寫** 到左邊的代碼區，無需手動尋找。

✨ **新功能：字體微調**
在 Tweaker 中使用 `+` / `-` 按鈕，系統會自動在 Markdown 中插入 `{size=24}` 標籤，精準控制個別元素的大小。

### ⌨️ Command Palette (指令面板)
按下 `Ctrl + K` (Windows) 或 `Cmd + K` (Mac) 喚起指令中心。
- **🔍 搜尋**: 輸入關鍵字，快速跳轉到特定投影片。
- **➕ 插入**: 快速插入圖表模板、Lorem Ipsum 假文。
- **🌗 模式**: 切換深色/淺色編輯器介面。
- **📤 匯出**: 執行 PPTX 下載或圖片包匯出。

## 7. 專業演講：雙螢幕與遙控

MD2PPT 不只是編輯器，更是強大的演講工具。

### 啟動演講模式
點擊 Header 的 **Present** 按鈕：
1.  **觀眾視窗 (Audience View)**：會在新視窗彈出，請將其拖曳到投影機螢幕並全螢幕。
2.  **演講者主控台 (Console)**：留在您的筆電螢幕，顯示下一頁預覽、備忘錄與計時器。

### 📱 P2P 手機遙控器
無需安裝 App，無需連上同一個 Wi-Fi。
1.  在演講者主控台點擊 **Mobile Remote**。
2.  用手機掃描螢幕上的 **QR Code**。
3.  您的手機即刻變身簡報筆！支援翻頁與黑屏 (Blackout) 功能。
